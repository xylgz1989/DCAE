# DCAE Project

This project uses the Disciplined Consensus-Driven Agentic Engineering framework.

## BMAD Workflow

This project follows the BMAD (Business, Architect, Developer) workflow:

- Business: Requirements and planning
- Architect: System design and architecture
- Developer: Implementation

## Getting Started

1. Configure your LLM providers in `.dcae/config.yaml`
2. Run `dcae run` to start the workflow
